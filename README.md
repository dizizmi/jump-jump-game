# jump-jump-game

A python game using Pygame library. 

Source for assets: 
- https://kenney.nl/assets/platformer-art-pixel-redux
- russs123 platform game

# Install

1. Clone git repo
```
https://github.com/dizizmi/jump-jump-game
```

2. Run platformer.py

3. Play game :)
